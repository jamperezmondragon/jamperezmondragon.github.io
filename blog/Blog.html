Segment Tree Beats

Segment Tree Beats es una forma de extender un Segment Tree para resolver problemas bastante difíciles. Fue inventada hace unos años por unos compas en el proceso a la IOI de china (:rofl:).


        #include <bits/stdc++.h>
        using namespace std;
        
        int main() {
          cout << "Hello, world!" << endl;
          return 0;
        }

La idea principal es extender las condiciones que te indican si entrar o no a un nodo en el segtree, así como definir un monovariante que siempre disminuye o aumenta con cada update. A este monovariante se le suele conocer como función potencial. Bueno, el punto esq el potencial está acotado, y por lo tanto la complejidad amortizada del algoritmo es rapidita.
